wget https://rubygems.org/rubygems/rubygems-2.6.8.tgz

cd rubygems-2.6.8
ruby setup.rb
安装jekyll
gem install jekyll
jekyll new myblog error
解决方法:
Dependency Error: Yikes! It looks like you don’t have kramdown or one of its
dependencies installed. In order to use Jekyll as currently configured,
you’ll need to install this gem. The full error message from Ruby is:
‘cannot load such file – kramdown’ If you run into trouble,
you can find helpful resources at http://jekyllrb.com/help/! gem install kramdown
解决办法 :
bundle exec jekyll serve --watch
在之前的命令 jekyll serve --watch 加上bundle exec . 因为我使用了 bundler 管理了gems.
